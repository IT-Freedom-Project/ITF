# Настройка проксирование UDP через iptables (Ubuntu/Debian)

Скрипт в режиме диалога настраивает сеть для проксирования UDP трафика, текущие правила можно смотреть, а также полностью их удалять.

Для запуска скрипта выполняем эту команду:\
\
```sudo wget -O udp-proxy.sh hhttps://raw.githubusercontent.com/IT-Freedom-Project/ITF/refs/heads/main/proksirovanie-trafika/proksirovanie-udp/udp-proxy.sh && sudo bash udp-proxy.sh```\
\
или эту: \
\
```sudo curl -o udp-proxy.sh https://raw.githubusercontent.com/IT-Freedom-Project/ITF/refs/heads/main/proksirovanie-trafika/proksirovanie-udp/udp-proxy.sh && sudo bash udp-proxy.sh```\
